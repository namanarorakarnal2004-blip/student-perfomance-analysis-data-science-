# Customer Segmentation for Targeted Online Advertising

## 📌 Objective
To segment customers based on behavior and demographics to improve targeted advertising.

## 🛠 Tools Used
- Excel
- Google Analytics (conceptual)
- Data Analysis

## 📊 Approach
- Data Collection
- Data Cleaning
- Segmentation:
  - Demographic
  - Geographic
  - Psychographic
  - Behavioral

## 🔍 Key Insights
- Targeted ads increase conversion rate
- Customer segmentation reduces marketing cost
- Personalized ads improve customer satisfaction

## 💡 Real Examples
- Amazon: Recommendation system
- Netflix: Personalized content
- Zomato: Location-based targeting

## 📈 Outcome
Improved understanding of how businesses use data analytics for targeted marketing.
